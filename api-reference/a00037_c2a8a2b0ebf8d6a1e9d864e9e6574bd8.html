los_membox.h
UINT32 LOS_MemboxInit ( VOID *  pBoxMem,
UINT32  uwBoxSize,
UINT32  uwBlkSize  
)

Initialize a memory pool.

Description:
  • This API is used to initialize a memory pool.
注意:
  • The uwBoxSize parameter value should match the following two conditions : 1) Be less than or equal to the Memory pool size; 2) Be greater than the size of LOS_MEMBOX_INFO.
参数:
参数名称 描述
pBoxMem  [IN] Memory pool address.
uwBoxSize  [IN] Memory pool size.
uwBlkSize  [IN] Memory block size.
返回值:
返回值 描述
LOS_NOK  The memory pool fails to be initialized.
LOS_OK  The memory pool is successfully initialized.
Dependency:
  • los_membox.h: the header file that contains the API declaration.
参见:
None.
