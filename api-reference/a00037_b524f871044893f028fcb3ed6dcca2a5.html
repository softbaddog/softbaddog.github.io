los_membox.h
UINT32 LOS_MemboxFree ( VOID *  pBoxMem,
VOID *  pBox  
)

Free a memory block.

Description:
  • This API is used to free a memory block.
注意:
  • The input pPool parameter must be initialized via func LOS_MemboxInit.
  • The input pBox parameter must be allocated by LOS_MemboxAlloc.
参数:
参数名称 描述
pBoxMem  [IN] Memory pool address.
pBox  [IN] Memory block address.
返回值:
返回值 描述
LOS_NOK  This memory block fails to be freed.
LOS_OK  This memory block is successfully freed.
Dependency:
  • los_membox.h: the header file that contains the API declaration.
