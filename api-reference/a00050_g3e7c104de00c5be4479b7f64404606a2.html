atiny_cmd_ioctl
Agenttiny Definition
int atiny_cmd_ioctl ( atiny_cmd_e  cmd,
char *  arg,
int  len  
)

issue the command.

Description:
This API is used to issue the command.
注意:
none.
参数:
参数名称 描述
cmd  [IN] The command to be issued. atiny_cmd_e.
arg  [IN/OUT] Buffer to store the command parameters. Agent_tiny is responsible for memory allocation. if cmd is GET_XXX, arg is outcoming parameter and this buffer is filled by device; else arg is incoming parameter and this buffer is filled by agent_tiny.
len  [IN] The length of the argument.
返回值:
返回值 描述
int  0 if succeed, or the error number atiny_error_e if failed.
Dependency: none.
参见:
none.
