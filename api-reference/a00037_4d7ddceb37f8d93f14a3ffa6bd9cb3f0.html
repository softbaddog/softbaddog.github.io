los_membox.h
UINT32 LOS_MemboxStatisticsGet ( VOID *  pBoxMem,
UINT32 *  puwMaxBlk,
UINT32 *  puwBlkCnt,
UINT32 *  puwBlkSize  
)

calculate membox information.

Description:
  • This API is used to calculate membox information.
注意:
  • One parameter of this interface is a pointer, it should be a correct value, otherwise, the system may be abnormal.
参数:
参数名称 描述
pBoxMem  [IN] Type VOID* Pointer to the calculate membox.
uwMaxBlk  [OUT] Type UINT32* Record membox max block.
uwBlkCnt  [OUT] Type UINT32* Record membox block count alreay allocated.
uwBlkSize  [OUT] Type UINT32* Record membox block size.
返回值:
返回值 描述
LOS_OK  The heap status calculate success.
LOS_NOK  The membox status calculate with some error.
Dependency:
  • los_memory.h: the header file that contains the API declaration.
参见:
LOS_MemAlloc | LOS_MemRealloc | LOS_MemFree
