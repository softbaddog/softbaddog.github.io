los_memory.h
UINT32 LOS_MemNodeSizeCheck ( VOID *  pPool,
VOID *  pPtr,
UINT32 *  puwTotalSize,
UINT32 *  puwAvailSize  
)

Check the size of memory node specified.

Description:
  • This API is used to check the size of memory node.
注意:
  • The input pPool parameter must be initialized via func LOS_MemInit.
  • The input pPtr parameter must be allocated by LOS_MemAlloc or LOS_MemAllocAlign.
  • The function will be called by function specified, such as memset or memcpy.
  • The feature can be enabled when you set the macro value of LOSCFG_BASE_MEM_NODE_SIZE_CHECK as YES.
  • You had better set memory check level as LOS_MEM_CHECK_LEVEL_DISABLE when copy bin file.
参数:
参数名称 描述
pPool  [IN] A pointer pointed to the memory pool.
pPtr  [IN] A pointer pointed to the source node.
puwTotalSize  [OUT] A pointer to save total size, must point to valid memory.
puwAvailSize  [OUT] A pointer to save available size, must point to valid memory.
返回值:
返回值 描述
OS_ERRNO_MEMCHECK_DISABLED  Memcheck function does not open.
OS_ERRNO_MEMCHECK_NOT_INIT  Memcheck function does not init.
OS_ERRNO_MEMCHECK_PARA_NULL  The pool or pPtr is NULL.
OS_ERRNO_MEMCHECK_OUTSIDE  The pPtr address is not in the reasonable range.
OS_ERRNO_MEMCHECK_NO_HEAD  Can't find the control head node from pPtr.
OS_ERRNO_MEMCHECK_WRONG_LEVEL  The memory check level is illegal.
LOS_OK  Success to get total size and available size of the memory node (pPtr).
Dependency:
  • los_memory.h: the header file that contains the API declaration.
