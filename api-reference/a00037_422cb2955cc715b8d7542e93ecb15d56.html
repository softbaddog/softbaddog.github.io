los_membox.h
VOID* LOS_MemboxAlloc ( VOID *  pBoxMem  ) 

Request a memory block.

Description:
  • This API is used to request a memory block.
注意:
  • The input pPool parameter must be initialized via func LOS_MemboxInit.
参数:
参数名称 描述
pBoxMem  [IN] Memory pool address.
返回值:
返回值 描述
VOID*  The request is accepted, and return a memory block address.
NULL  The request fails.
Dependency:
  • los_membox.h: the header file that contains the API declaration.
