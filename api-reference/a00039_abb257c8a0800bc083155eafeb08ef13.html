los_memory.h
UINT32 LOS_MemFreeBlksGet ( VOID *  pPool  ) 

Get the number of free memory nodes.

Description:
  • This API is used to get the number of free memory nodes in memory pool.
注意:
  • The input pPool parameter must be initialized via func LOS_MemInit.
参数:
参数名称 描述
pPool  [IN] A pointer pointed to the memory pool.
返回值:
返回值 描述
LOS_NOK  The incoming parameter pPool is NULL.
UINT32  The number of free memory nodes.
Dependency:
  • los_memory.h: the header file that contains the API declaration.
参见:
None.
