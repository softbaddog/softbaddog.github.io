los_memory.h
UINT32 LOS_MemCheckLevelSet ( UINT8  ucLevel  ) 

Set the memory check level.

Description:
  • This API is used to set the memory check level.
注意:
  • There are three level you can set.
  • The legal level are LOS_MEM_CHECK_LEVEL_LOW, LOS_MEM_CHECK_LEVEL_HIGH, LOS_MEM_CHECK_LEVEL_DISABLE.
参数:
参数名称 描述
ucLevel  [IN] The level what you want to set.
返回值:
返回值 描述
LOS_ERRNO_MEMCHECK_WRONG_LEVEL  The memory check level what you want to set is illegal.
LOS_OK  Success to set the memory check level.
Dependency:
  • los_memory.h: the header file that contains the API declaration.
