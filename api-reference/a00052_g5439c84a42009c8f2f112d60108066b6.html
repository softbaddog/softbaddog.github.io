atiny_net_connect
Agenttiny Socket
void* atiny_net_connect ( const char *  host,
const char *  port,
int  proto  
)

create socket and connect to the server

Description:
This API is used to create socket and connect to the server. It will do name resolution with both IPv6 and IPv4, do internal memory allocation for socket structure and release it by the atiny_net_close interface.
注意:
none.
参数:
参数名称 描述
host  [IN] Host to connect to.
port  [IN] Port to connect to.
proto  [IN] Protocol: ATINY_PROTO_TCP or ATINY_PROTO_UDP.
返回值:
返回值 描述
pointer  Point to the Socket you created.
NULL  Create socket or connect to server failed.
Dependency: none.
参见:
atiny_net_recv | atiny_net_send | atiny_net_recv_timeout | atiny_net_close
