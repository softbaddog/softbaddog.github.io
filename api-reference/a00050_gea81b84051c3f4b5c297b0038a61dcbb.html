atiny_data_report
Agenttiny Definition
int atiny_data_report ( void *  phandle,
data_report_t *  report_data  
)

report the data to the server.

Description:
This API is used to report the data to the server.
注意:
none.
参数:
参数名称 描述
phandle  [IN] The handle of the agent_tiny.
report_data  [IN] Data to be reported.
返回值:
返回值 描述
ATINY_OK  Report succeed.
ATINY_MSG_CONGEST  The lwm2m protocal stack is congested. It is occured when the number of report messages are greater than storing_cnt in atiny_server_param_t.
ATINY_BUF_OVERFLOW  The length of the buffer in report_data is exceeded MAX_REPORT_DATA_LEN.
Dependency: none.
参见:
none.
